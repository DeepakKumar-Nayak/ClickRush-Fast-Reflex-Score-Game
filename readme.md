## “ClickRush – Fast Reflex Score Game”
A simple browser-based reaction game built using HTML, CSS, and JavaScript.

The goal is simple:
👉 Click the moving box as fast as possible and increase your score.

⸻

🚀 Features

* Real-time score tracking
* Moving box with random positions
* Dynamic color changes
* Timer system
* Click sound effect
* Game over sound effect
* Anti-spam click protection (cooldown system)
* Start / Stop game controls
* Unnecessary clicks are ignored (box clicks are counted only during valid game logic controlled by timer system)

⸻

🧠 What I learned

* DOM manipulation in real-time
* Game loop logic using setInterval
* Event handling optimization
* State management (score, time, game states)
* Audio integration in web apps
* Basic browser storage concepts


⸻

🛠️ Tech Stack

* HTML
* CSS
* JavaScript (Vanilla)

⸻

💡 Key Highlight

A simple project, but it helped me understand how real-time browser games work — especially how UI, logic, and state interact together.

This project is built purely for practice and learning purposes.

Still improving step by step 🚀
